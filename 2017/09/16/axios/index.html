前言

Axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中。这个是官网的描述，最近用到了vue全家桶来进行开发，里面也集成了这个工具，它的作用是用来进行http请求的，对前端开发来说，那这个就是我们跟后台交互的方式了啊，那么就开始学吧…

项目实例

1、 首先我们要在项目中安装 axios，在项目根目录打开命令行，键入 npm install axios --save 

- 执行了这个命令后会再 package.json 文件中记录下这个依赖

2、在目录 src 底下新建个 http 的文件夹，http文件夹底下新建个 index.js

3、调一个接口试试，打开你需要 http 请求的 vue 文件
