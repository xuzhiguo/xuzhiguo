Vue全家桶+SSR+Koa2全栈开发美团网 遇到的一些问题

xuzg 发表于 2019-07-06

1. 如何在 Koa2 中用es6的 import 代替 require 这种引入方式

1) 安装 babel-cli
2) 把 package.json 中的 start 改为 cross-env NODE_ENV=production babel-node server/index.js

2. 启动 mongoDB 时报错 Data directory D:\data\db\ not found., terminating

  • 应该是没有指定数据库保存目录的原因，执行以下命令：
    mongod -dbpath <你想保存的目录>
    例如：mongod -dbpath D:\Program Files\MongoDB\Server\4.0\data\db

3. redis 启动 报错 Warning: no config file specified, using the default config. In order to specify a config file use redis-server /path/to/redis.conf

  • 感觉是 redis 是已经启动了，但是假死状态（连不上…），执行下面命令，关闭重启就行了

4. npm run dev 启动项目时提示 BrowserslistError: Unknown browser query android all

  • 感觉是 browserslist 版本更新的问题，我重装了依赖后，就开始报这个警告
  • github 上 browserslist 的 Issues 链接
  • 按照 Issues 上大佬的提示，执行以下步骤，解决了
